## Methods Directory Overview

The `methods/` directory contains implementations of the four estimation strategies used in this study. Each subdirectory corresponds to a specific methodological approach and includes the associated Python code and any generated output data (e.g., CSV files).

Do not that:
- Each subdirectory is self-contained and may rely on shared upstream datasets (e.g., simulation grids or sampled patient data).
- Implementations may differ slightly in preprocessing, filtering, or evaluation details depending on the method.
