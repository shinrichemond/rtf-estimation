## rtf-estimation/thyroSim Source Reference

This folder includes selected components from the original Thyrosim model, which serves as the foundational reference for the Python implementation.

The original ThyroSim model, developed by the DiStefano Lab, was **not used directly in this project**. Instead, it was reviewed and selectively referenced to guide the process of porting the model into Python while maintaining consistency with the underlying physiological logic and parameterization.

**Original Source Repository:**  
https://bitbucket.org/DistefanoLab/thyrosim

**Date Retrieved:**  
March 7, 2026

## Imported Components

Only a subset of the original repository was incorporated for reference purposes:

- **Parameters**  
  Used to replicate model constants and ensure numerical consistency with the original implementation.

- **Java Folder**  
  Reviewed to understand the structure and execution flow of the original model.

- **ThyroSim Code**  
  Referenced for translating core model equations and logic into Python.

## Purpose of Inclusion

These components were included strictly for:

- Cross-referencing implementation details  
- Verifying correctness during the porting process  
- Preserving alignment with the original model design  

They are **not executed as part of the current system** and serve only as a static reference.
