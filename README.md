# rtf-estimation

A simulation-based framework for estimating Residual Thyroid Function (RTF) from clinically measurable hormone levels (FT4, FT3, TT3, TSH) in the presence of exogenous hormone therapy.

RTF represents the remaining endogenous hormone production capacity of the thyroid gland. While RTF can be inferred in untreated patients using forward simulation, estimation becomes significantly more difficult under hormone replacement therapy due to confounding effects from exogenous LT4 and LT3.

This project develops and evaluates multiple estimation strategies based on the p-THYROSIM model (https://doi.org/10.3389/fendo.2025.1735282).
